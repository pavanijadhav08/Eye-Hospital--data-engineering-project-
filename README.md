# Eye Hospital Data Engineering Project

This project focuses on building a complete data engineering pipeline for healthcare data collected from EMR systems and medical camps. The goal is to transform raw, unstructured Excel data into a clean, structured, and automated SQL database system.


## Objectives
- Collect healthcare data from multiple sources (EMR & medical camps)
- Perform data profiling to identify quality issues
- Clean and standardize raw datasets
- Design a normalized SQL database schema
- Build an automated ETL pipeline using Python
- Ensure data accuracy through validation, testing, and logging

## Tech Stack
- Python (Pandas, Data Processing)
- SQL (Database Design & Queries)
- Excel (Raw Data Source)
- ETL Pipeline (Extract, Transform, Load)

## ETL Pipeline

- The project follows a structured ETL process:

- Extract – Data collected from Excel files
- Transform – Cleaning, formatting, and validation using Python
- Load – Data inserted into a SQL database

## Data Cleaning Steps
- Removed missing/null values
- Eliminated duplicate records
- Standardized date formats
- Fixed inconsistent text entries
- Structured columns properly


## Automation & Testing
- Automated ETL process using Python scripts
- Implemented:
   - One-click data loading
   - Batch data insertion
   - Error handling
- Validation:
   - Row count checks
   - Data load verification
- Logging for monitoring and traceability


## Key Outcomes
- Clean and structured datasets
- Centralized SQL database
- Automated ETL pipeline
- Improved data quality and consistency
- Reduced manual effort
